1. Napisz program, który pobierze od użytkownika imię i nazwisko ucznia, a następnie dwie oceny. Oceny są liczbami całkowitymi. Program wyświetla imię i nazwisko ucznia oraz jego średnią ocen.

    Przykład działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj pierwszą ocenę: 3
                Podaj drugą ocenę: 4
                Jan Kowalski, średnia ocen: 3.5

  2. Popraw program tak, żeby wymuszał podanie oceny z zakresu 1–6, tzn. żeby prosił do skutku o podanie poprawnej oceny.

    Przykład działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj pierwszą ocenę: -1
                Podaj liczbę z zakresu 1-6: 0
                Podaj liczbę z zakresu 1-6: 7
                Podaj liczbę z zakresu 1-6: 3
                Podaj drugą ocenę: 4
                Jan Kowalski, średnia ocen: 3.5

  3. Popraw program tak, żeby najpierw pytał o liczbę ocen i wymuszał przy tym podanie liczby dodatniej.

    Przykład działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: -1
                Podaj liczbę dodatnią: 0
                Podaj liczbę dodatnią: 4
                1. ocena: 2
                2. ocena: -1
                Podaj liczbę z zakresu 1-6: 0
                Podaj liczbę z zakresu 1-6: 7
                Podaj liczbę z zakresu 1-6: 3
                3. ocena: 4
                4. ocena: 6
                Jan Kowalski, średnia ocen: 3.75

  4. Popraw program tak, żeby zamiast średniej wyświetlał ocenę końcową. Ocena końcowa jest liczbą całkowitą — zaokrągloną w górę średnią.

    Przykład działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: -1
                Podaj liczbę dodatnią: 0
                Podaj liczbę dodatnią: 4
                1. ocena: 2
                2. ocena: -1
                Podaj liczbę z zakresu 1-6: 0
                Podaj liczbę z zakresu 1-6: 7
                Podaj liczbę z zakresu 1-6: 3
                3. ocena: 4
                4. ocena: 6
                Jan Kowalski, ocena końcowa: 4

  5. Popraw program tak, żeby program wyświetlał również informację o tym, czy uczeń zdał (ocena końcowa większa niż 1), czy nie (ocena końcowa równa 1).

    Przykłady działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 2
                1. ocena: 2
                2. ocena: 3
                Jan Kowalski, ocena końcowa: 3, uczeń zdał.
            
                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 3
                1. ocena: 1
                2. ocena: 1
                3. ocena: 2
                Jan Kowalski, ocena końcowa: 1, uczeń nie zdał.

  6. Popraw program tak, żeby umożliwiał podanie zera jako liczby ocen. Natomiast w takiej sytuacji oprócz imienia i nazwiska wyświetlana jest wyłącznie informacja o tym, że uczeń jest nieklasyfikowany.

    Przykłady działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 2
                1. ocena: 2
                2. ocena: 3
                Jan Kowalski, ocena końcowa: 3, uczeń zdał.
            
                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 3
                1. ocena: 1
                2. ocena: 1
                3. ocena: 2
                Jan Kowalski, ocena końcowa: 1, uczeń nie zdał.
            
                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 0
                Jan Kowalski, uczeń nieklasyfikowany.

  7. Popraw program tak, żeby po pobraniu danych ucznia pytał użytkownika o to, czy chce podać dane kolejnego (dane pierwszego ucznia pobierane są zawsze).

    Przykład działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 2
                1. ocena: 2
                2. ocena: 3
                Jan Kowalski, ocena końcowa: 3, uczeń zdał.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? t
                Imię: Anna
                Nazwisko: Nowak
                Podaj liczbę ocen: 3
                1. ocena: 1
                2. ocena: 1
                3. ocena: 2
                Anna Nowak, ocena końcowa: 1, uczeń nie zdał.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? t
                Imię: Piotr
                Nazwisko: Wójtowicz
                Podaj liczbę ocen: 0
                Piotr Wójtowicz, uczeń nieklasyfikowany.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? n

  8. Popraw program tak, żeby na końcu wyświetlał informację o tym ilu uczniów zdało, ilu nie zdało i ilu jest nieklasyfikowanych.

    Przykład działania: 

                    Imię: Jan
                Nazwisko: Kowalski
                Podaj liczbę ocen: 2
                1. ocena: 2
                2. ocena: 3
                Jan Kowalski, ocena końcowa: 3, uczeń zdał.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? t
                Imię: Anna
                Nazwisko: Nowak
                Podaj liczbę ocen: 3
                1. ocena: 1
                2. ocena: 1
                3. ocena: 2
                Anna Nowak, ocena końcowa: 1, uczeń nie zdał.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? t
                Imię: Piotr
                Nazwisko: Wójtowicz
                Podaj liczbę ocen: 0
                Piotr Wójtowicz, uczeń nieklasyfikowany.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? t
                Imię: Katarzyna
                Nazwisko: Grzegorczyk
                Podaj liczbę ocen: 2
                1. ocena: 5
                2. ocena: 5
                Katarzyna Grzegorczyk, ocena końcowa: 5, uczeń zdał.
                Czy chcesz podać oceny kolejnego ucznia (t/n)? n
                Liczba uczniów, którzy zdali: 2
                Liczba uczniów, którzy nie zdali: 1
                Liczba uczniów nieklasyfikowanych: 1

  9. Popraw program z zadania piątego tak, żeby dane pobierał z pliku tekstowego. Pierwszy wiersz pliku zawiera imię i nazwisko, drugi wiersz — wyłącznie liczbę ocen, trzeci wiersz — oceny.

    Przykładowa zawartość pliku z danymi: 

                    Jan Kowalski
                3
                1 1 2

    Program wyświetla informacje takie same jak w zadaniu piątym, np. dla powyższego pliku powinien wyświetlić: 

                    Jan Kowalski, ocena końcowa: 1, uczeń nie zdał.

    W przypadku błędnych danych (liczba ocen nie jest dodatnia, któraś z ocen jest spoza zakresu) program wyświetla tylko o tym informację i kończy działanie.
